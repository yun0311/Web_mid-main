HTML(Hypertext Markup Language)

웹페이지를 만드는데 사용되는 언어

Hypertext는 웹 페이지를 다른 페이지로 연결하는 링크를 말한다.

공식 문서
명령어/태그 설명
<h1> 가장 중요한 제목을 나타냅니다.
<p> 단락을 나타냅니다.
<a> 하이퍼링크를 만듭니다.
<img> 이미지를 삽입합니다.
<ul>, <li> 리스트와 리스트 항목을 만듭니다.

예시 코드

<h1>Hello</h1>

출력값

Hello

CSS(Cascading Style Sheets)

CSS는 요소가 화면, 종이, 음성이나 다른 매체 상에 어떻게 렌더링되어야 하는지 지정합니다.

공식 문서
명령어/속성 설명
color 텍스트의 색상을 지정합니다.
background-color 요소의 배경색을 지정합니다.
font-size 텍스트의 크기를 지정합니다.
margin 요소의 외부 여백을 지정합니다.
display 요소의 박스 유형(블록, 인라인 등)을 지정합니다.

예시 코드

h1 {
  color: blue;
  background-color: #f0f0f0;
  font-size: 32px;
  margin: 20px 0;
  display: block;
}

출력값

Hello

JavaScript

웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.

공식 문서
명령어/문법 설명
let, const, var 변수를 선언할 때 사용합니다.
function 함수를 정의할 때 사용합니다.
if, else 조건문을 작성할 때 사용합니다.
for, while 반복문을 작성할 때 사용합니다.
console.log() 콘솔에 값을 출력할 때 사용합니다.

예시 코드

let message = "Hello, JavaScript!";
if (message) {
  console.log(message);
}

출력값

Hello, JavaScript!

VSCode

VSCode(Visual Studio Code)는 마이크로소프트에서 개발한 무료 오픈 소스 코드 편집기입니다. 다양한 프로그래밍 언어를 지원하며, 가볍고 빠른 성능과 풍부한 확장 기능, 직관적인 UI로 많은 개발자들이 선호하는 개발 도구입니다. 확장 프로그램을 통해 디버깅, 버전 관리, 테마 등 다양한 기능을 추가할 수 있습니다.

Next.js

Next.js는 React 기반의 오픈 소스 프레임워크로, 서버사이드 렌더링(SSR), 정적 사이트 생성(SSG), 라우팅 등 현대적인 웹 애플리케이션 개발에 필요한 다양한 기능을 제공합니다. 빠른 페이지 로딩, SEO 최적화, 코드 분할 등 성능과 개발 편의성을 동시에 갖추고 있어 기업과 개인 개발자 모두에게 널리 사용되고 있습니다.
